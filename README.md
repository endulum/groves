# Groves
Groves is an arboreal clone of Reddit.

[Project Spec](https://www.theodinproject.com/lessons/node-path-nodejs-odin-book)

## Technologies used
- **Lang:** React with Typescript + TSX
- **Bundler:** Vite
- **Linting:** ESLint with the Airbnb style guide and React recommendations
